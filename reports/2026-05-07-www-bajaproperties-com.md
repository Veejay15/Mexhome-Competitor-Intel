## 3. Backlink Movements

The SEMrush backlinks CSV for Baja properties contains 295 rows. All links in the top rows are flagged as new this week.

**Pattern: Spam link cluster targeting the homepage.** The dominant signal across the dataset is a cluster of zero-authority domains (Page Authority Score: 0) all running pages with an identical title format: "Baja Properties - Cabo Real Estate Agents Address, Contact Info etc." Domains include ranaalper.com, chrislydgate.com, spacultures.com, sorridi.net, raiski-kat.com, micronetbroadband.net, mein-solardach.com, railway-fastener.com, spikule.net, fizkunst.com, verlagsgesellschaft.net, michaelscheiner.net, and a dozen more. All link to bajaproperties.com using the homepage URL as anchor text, have only 3 external links and 3 internal links, are dofollow, and were first seen within a 3-day window (May 4 to May 7). This is a coordinated spam link pattern. The SEO value is negligible and likely negative if Google has already identified the network.

**One legitimate new link.** Remitly (remitly.com, Authority Score: 18) linked to Baja properties' La Paz real estate blog post in a "How Much Does a House Cost in Mexico in 2026" guide, using the anchor "start around." This is an editorial citation in a high-intent buyer research article. It is the only link of meaningful quality in the top rows this week.

**Two additional low-quality directory links.** loscabosrealestatecom.rs5.aios-staging.com (a staging domain, Authority Score: 0) linked via an image, and bestwebstats.com (a domain-stats aggregator with 10,159 external links on the linking page, Authority Score: 0) included bajaproperties.com in what appears to be a bulk-stats page. Neither carries meaningful authority.

**No lost links** are recorded in the top rows.

**Action implication for MexHome:** The Remitly citation is worth monitoring. Remitly is covering Mexico property costs as a buyer resource. MexHome should seek similar editorial placements in personal finance and expat resource publications. The spam link cluster is not a threat to MexHome directly but confirms Baja properties is not running a clean, high-authority link-building program in Baja California at present.

Strengthen those existing pages with richer on-page content: updated listing counts, neighborhood breakdowns, buyer intent copy, and schema markup. Do not build new pages for these queries. Focus investment on making the existing pages more competitive.

2. **Strengthen the La Paz real estate page to compete with Baja properties' Remitly-cited blog.** Baja properties earned an editorial link from Remitly's "how much does a house cost in Mexico" buyer guide, citing their La Paz real estate blog at position 5 for "la paz mexico real estate" (1,300 monthly searches, KD 14). MexHome already has /area/los-cabos/la-paz/, /area/los-cabos/la-paz/la-paz-homes-for-sale/, /area/los-cabos/la-paz/la-paz-condos-for-sale/, and /area/los-cabos/la-paz/la-paz-land-for-sale/. Expand the main /area/los-cabos/la-paz/ page with pricing data, neighborhood summaries, and FAQ content that would make it citable by personal finance and expat publications. Then pursue outreach to similar resources covering Mexico property costs.

3. **Build a "buying property in Mexico as an American" guide if MexHome does not have an equivalent page.** Baja properties jumped from position 8 to position 2 for this keyword (590 monthly searches, KD 10). Cross-referencing MexHome's page list, there is no direct equivalent to this query as a standalone guide. MexHome does have /blog/buying-a-house-in-mexico/ and /blog/buying-property-in-mexico-a-comprehensive-guide-for-foreigners/. Review whether either of those pages explicitly addresses the "as an American" angle (fideicomiso, title, bank trust, restrictions). If the coverage is thin on that specific framing, update the most relevant existing post to target this query rather than creating a duplicate.

4. **Monitor Baja properties' "los barriles real estate" position as a potential gap.** Their page /los-barriles-area/ dropped from position 1 to position 5 for "los barriles real estate" (320 monthly searches, KD 15). MexHome has /area/los-cabos/east-cape/los-barriles/ as an area page. If that page has thin content, this is a low-difficulty keyword where an updated, content-rich area page could move quickly. MexHome's resources are better spent reinforcing transactional and commercial pages where Baja properties is actively losing ground.

6. **Disregard the spam backlink cluster as a competitive signal.** The coordinated zero-authority directory links pointing to bajaproperties.com this week carry no legitimate authority and do not represent a link-building approach worth emulating or feeling threatened by. Focus MexHome's link-building attention on editorial outreach to personal finance, expat, and real estate publications in the model of the Remitly citation.
