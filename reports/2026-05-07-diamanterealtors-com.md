## 3. Backlink Movements

SEMrush CSV data is available for this competitor. The backlinks file contains 123 rows. The following observations are drawn from the top rows provided.

**Overall profile quality: low.** The highest authority source in the sample is a Nigerian political news site (streetwiseworld.com.ng, Page Authority score 35) linking to their Loreto listings page with the anchor "loreto real estate." This is almost certainly spam or a link-farm placement, not an earned editorial link.

**The only link that carries any real weight:** Mexico News Daily (mexiconewsdaily.com, Page Authority 13) links to Diamante's fideicomiso explainer page with the anchor "fideicomiso." Mexico News Daily is a legitimate, widely-read English-language publication targeting the expat and buyer audience directly. This is a meaningful editorial placement.

**Additional observations from the sample:**

- Multiple links originate from YesWiki and Colibris collaborative wiki platforms (Page Authority scores 2 to 6), all using keyword-rich anchors such as "cabo real estate," "baja real estate," "la ventana real estate," and "loreto real estate." These are spam link patterns with no SEO value.
- luxuryrealestate.com (Page Authority 9) links to the Diamante homepage with a blank anchor. This is a legitimate real estate directory citation, though the authority is modest.
- A Mongolian news site (tsenzuur.mn, Page Authority 6), a Malaysian marketplace (murahkitchen.my, Page Authority 3), and a Croatian conservation project site (capusproject.eu, Page Authority 3) all link to Diamante property pages. These are clearly junk links with no topical relevance.
- overseasdreamhome.com (Page Authority 2) links to a specific East Cape listing with a descriptive anchor referencing a $599,000 property. This is likely a genuine editorial mention, though the domain authority is negligible.
- hotfrog.com.mx (Page Authority 6) lists Diamante under their old brand name "Diamante Associates," linking to the homepage with the anchor "www.diamanteassociates.com." This suggests a legacy business directory citation from a prior brand identity.

**Summary:** Diamante's backlink profile shows almost no high-quality, topically relevant link acquisition. The Mexico News Daily fideicomiso link is the exception. Their rankings in competitive Cabo and La Paz terms are unlikely to be sustained by this backlink foundation, which is consistent with the ranking losses observed in the positions data.
